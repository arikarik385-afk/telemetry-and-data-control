# telemetry-and-data-control
Веб - приложение, позволяющее отслеживать показания датчиков температуры, давления, расхода и заполненности установки.
# 📡 Мониторинг датчиков

Система мониторинга показаний датчиков с тепловой картой, лентой событий и админ-панелью.

## 🚀 Быстрый запуск (Docker)

### Требования
- Docker Desktop (с WSL2 на Windows)
- Git

### Установка и запуск

```bash
# 1. Клонировать репозиторий
git clone https://github.com/ТВОЙ_АККАУНТ/НАЗВАНИЕ_РЕПО.git
cd НАЗВАНИЕ_РЕПО

# 2. Запустить контейнеры
docker-compose up -d --build

# 3. Открыть в браузере
http://localhost:8501
